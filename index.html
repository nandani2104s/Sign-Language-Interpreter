Sign Language AI

A Computer Vision based interface for real-time American Sign Language recognition using MediaPipe and Machine Learning.

97.22% Accuracy 7 Gesture Classes

Project Overview

This project focuses on creating an accessible communication tool for the hearing-impaired community. By leveraging MediaPipe's Hand Landmark detection and a Random Forest Classifier, the system translates physical gestures into digital text in under 30ms, allowing for seamless real-time interaction.

01

Data Collection

Captured 1,400+ images (200 per class) across 7 gesture categories. Data was augmented by varying hand distance and rotation.

02

Feature Extraction

Used MediaPipe Task API to map 21 key landmarks. Coordinates were normalized to ensure the model focuses on shape rather than position.

x_norm = landmark.x - min(x_coordinates)
03

Model Training

An ensemble Random Forest model was trained on the landmark vectors. This provides high stability and handles non-linear patterns efficiently.

✓ 97.22% Precision Score
04

Live Inference

Integrated with OpenCV for real-time webcam feedback. The system overlays predicted labels and bounding boxes instantly.
